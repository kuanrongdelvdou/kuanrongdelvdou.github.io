3.Git 本地仓库基本操作

1.基本信息设置

    1> 设置用户名: git conifg --global user.name "kuanrongdelvdou"    
    
    

    2> 设置用户邮箱: git config --global user.email "kuanrongdelvdou@163.com"

    注意: 以上设置在 github 仓库主页显示谁提交了该文件

3.初始化一个仓库

    1>  新建一个文件夹
    

    2> 初始化 git (创建 git 仓库): git init

       [1]  进入创建好的文件夹 Git_temp_Repositry 文件夹

       [2] 使用 git init 命令将 Git_temp_Repositry 文件夹初始化为一个 git 仓库
       

       .git 文件: 是存储所有仓库信息的!


4.向仓库添加文件

    1> 先查看一下状态
    
    
    2> 新建一个 Student.java 文件: touch Student.java
    

    3> 继续查看状态
    

    4> 将 Student.java 文件添加到暂存区: git add Student.java
    

    5> 将暂存区的而文件提交到 git 仓库: git commit -m "提交信息"
    

5.修改仓库文件
    
    1> 修改 Student.java 文件内容
    

    2> 将文件添加到暂存区
    

    3> 将修改后的 Student.java 文件提交到仓库
    
    

6.删除仓库文件: git rm Student.java
    
    1> 删除工作区的 Student.java 文件
    

    2> 删除暂存区的 Student.java 文件: git rm Student.java
