1.ssh 链接

1.第一次安装好 git 与远程仓库建立 ssh 链接

    1> 生成 ssh key: kuanrongdelvdou@163.com 是你在 gitbub 上你注册的 邮箱地址!

       $ ssh-keygen -t rsa -b 4096 -C "kuanrongdelvdou@163.com"

    2> 把 ssh 添加到 github 上去
      
       复制文件c/Users/Administrator/.ssh/id_rsa.pub 内容，把key添加到：github > settings > SSH and GPG keys > New SSH key > 粘贴保存。

    3> 测试 ssh 链接是否成功
       
        $ ssh -T git@github.com

         

    4> 现在你可以通过 SSH 方式来 clone 及提交代码了
